## Tech Stack
Tools & Techniques used

* Java 1.8
* PostgreSQL
* Spring Boot
* Lombok
* JUnit/Mockito
* Angularjs
* Stomp


## How to run

- Install postgresql. Configuration is available in application.properties

- Hit below url:
  http://localhost:8080/

- Two Players will be created on Server Start up with below credentials

  First Player:
   Username - Player1
   Password - Player1

  Second Player:
   Username - Player2
   Password - Player2
